亲测最新可用美区 Apple ID 注册方法

发表于 2018-09-11 | 分类于 分享发现 | 阅读次数：
本文字数： 1.2k 字 | 阅读时长 ≈ 2 分钟

美区是 APP 数量最多的区，很多中国区没有或者下架了的 APP 美区都有，且有些 APP 仅限美区id限免！

可以说，拥有一个美区 Apple ID 是高玩的一个标配。毕竟世界那么大，谁不想去看看呢？

曾几何时，美区 Apple ID 注册不用绑信用卡，几秒钟搞定就可以下 APP 。

但是现在到了付款那里，就是看不到 「 None 」 这个选项！

这究竟是怎么一回事？？？

原来是苹果对访问 APP store 的 IP 地址设置了权限，如果是中国的 IP 地址就没有 「 None 」 这个选项……

看来最近苹果对跨区注册 ID 的限制越来越严格了，建议大家还是尽早注册一个，有备无患啊。

其实思考一下，应对方法很简单，只需要两步！

一、注册好一个美区 Apple ID ；

二、开启 VPN ，再登陆 APP Store 进行激活 。

完美解决！那么下面详细步骤分享给小白同学~

注册一个美区 Apple ID

浏览器打开 https:// Appleid. Apple.com/account#!&page=create 进入注册界面，地区选择“美国”，广告X掉，其他填好就继续下一步。

输入收到的验证码。

这里账号其实就已经注册好了，但为了方便后面登入 APP Store 后不用再填信息我们往下拉，有个付款方式。

点下添加付款方式。

填写美国的地址、邮编、电话。

这里推荐一个神奇的网站 http://www.fakeaddressgenerator.com/ ，点开这个网址，只需点一下 Random Address ，它立马就给你整一个假老美个人信息出来。

往下拖，邮编、地址、电话全出来了，就连银行卡信息都有！(别想了当然也是假的了)。

接着，只需要把地址电话这些信息补上，确保地区选的美国，付款方式选” None “，然后就点击存储吧！

到这里，注册美区 Apple ID 算是搞定了！后面就剩去 iPhone\iPad 激活账号下载 APP 了！

激活美区 Apple ID

先随便下载一个 VPN 工具，并打开使用。

用新注册的美区 Apple ID 登陆 APP Store ，随便搜一个 APP ，点 Get 就会弹出检验提示框。

点 Review 开始激活账号。

一直点击“下一步”，你懂的。

你会惊奇的发现交易方式那边出现了 None ！ 毫不犹豫勾上 None ，继续下一步。

搞定！那么就开始疯狂下载模式吧！
