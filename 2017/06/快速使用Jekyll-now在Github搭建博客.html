快速使用Jekyll-now在Github搭建博客

发表于 2017-06-13 | 分类于 分享创造 | 阅读次数：
本文字数： 533 字 | 阅读时长 ≈ 1 分钟

本来想学会用Jekyll和Hexo在Github搭建博客的，然后不出意外的困在本地部署上了。

再然后瞎搜了搜，就发现了可以在线编辑的利器 —— Jekyll-now !

不过…… 通过勤查苦试我还是学会了 Hexo ，所以分享下吧，也许有人需要呢。

Jekyll-now主要功能：

  • 响应式以及移动客户端优化的主题
  • Markdown 支持
  • 语法高亮
  • 集成 Disqus 评论
  • 集成 Google Analytics 统计
  • 底部提供 SVG 格式的社会化图标
  • 极少的 http 请求
  • 无需安装依赖
  • 无需设置本地开发环境
  • 无需配置插件
  • 无需花费时间在主题上

Jekyll-now部署步骤：

  1. 前往 Jekyll-now Fork项目 ；
  2. 在设置里将项目名称改为 yourgithubusername.github.io ，如：berwyns.github.io ；
  3. 在 _config.yml 文件里修改博客名称、描述以及头像 ；
  4. 开始访问你的博客，yourgithubusername.github.io 。
